
# Bruteay

Bruteay is a password wordlist generator 


## Authors

- [@Ayushman Dutta](https://www.github.com/ayushman89)


## to run the brtueay

Clone the project

```bash
  git clone https://github.com/Ayushman89/Bruteay.git
```

Go to the project directory

```bash
  cd Bruteay
```

ls

```bash
  ls Bruteay
```

To start the tool

```bash
  python3 Bruteay.py
```
